
# Seattle Energy Project

Projet de data science réalisé dans le cadre de la mission **« Anticipez les besoins en consommations de bâtiments »**.

L’objectif est de prédire la **consommation énergétique** de bâtiments non résidentiels de la ville de Seattle à partir de leurs caractéristiques structurelles, fonctionnelles, temporelles et géographiques.

## Objectifs du projet

- réaliser une **analyse exploratoire** du jeu de données ;
- construire des variables pertinentes via du **feature engineering** ;
- préparer les données pour la modélisation ;
- comparer plusieurs modèles supervisés de régression ;
- identifier les facteurs les plus influents sur la consommation énergétique.

## Données utilisées

Le projet s’appuie principalement sur le fichier :

- `2016_Building_Energy_Benchmarking.csv`
